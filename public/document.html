Documentation for Arty Movies API

-- now with CSS! --

Endpoint Description Method URL Body Response
Returns a JSON object of ALL movies GET /movies - JSON
Returns a JSON object of a single movie GET /movies/:title - JSON
Returns a JSON object of all movies of a certain genre GET /movies/genres/:genre - JSON
Returns a JSON object of a movie-director by name GET /directors/:name - JSON
Creates a new user, then gives acknowledgement POST /users JSON 201
Updates the username of a user, then gives acknowledgement PUT /users/:id/:username - 201
Adds a movie to a user's list of "favorites", then gives an acknowledgement POST /users/:id/:title - 201
Removes a movie from a user's list of favorites, then gives an acknowledgement DELETE /users/:id/:title - 201
Removes an existing user from the database DELETE /users/:id - 201
